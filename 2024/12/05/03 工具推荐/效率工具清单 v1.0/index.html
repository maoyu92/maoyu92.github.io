效率工具清单 v1.0

前言

总结工作以来觉得好用的工具和网站，我选择原则一是工作中高频应用，二是提高工作效率，从中受益匪浅的工具，三是开源免费，成本比较低的软件。列出来主要是方便自己查询，并让需要的人少走弯路。
随着时间推移，软件工具也在迅速迭代。因此，我会定期修改更新这份清单，以保持这份清单的时效性和实用性。注意排名有先后，全是个人喜好，请谨慎查看。

笔记与写作

名称 推荐语
Obsidian 最好用的笔记软件，支持双链、扩展性强，五星好评。
语雀 多端同步，功能齐全
Flomo 碎片笔记，obsidian 中 Thino 可平替
FlowUs 优点在免费公开文档
Mdnice Markdown 排版

效率工具

名称 推荐语
Quicker 快捷呼出的工具箱，插件市场强大，常用功能：批量重命名，拾色器，ever 智识、截图..
Listary 最好用的搜索启动工具，一直被模仿，从未被超越。
Everything 经典全盘搜索工具
OneCommander 开源资源管理器
Quicklook 预览软件，丰富插件，空格就能预览图片、文档、视频…
iTab 好用的浏览器启动页插件
滴答清单 待办清单软件
360 好压 当bandzip 出现广告的时候，你就是最棒的压缩软件
Windows 便笺 拒绝一切花里胡哨
TreeSize 快速查看文件夹组织结构与大小

阅读

名称 推荐语
Adobe Acrobat 功能齐全的 pdf 处理器
Zotero 开源文献管理工具，扩展性强
微信读书 出版书齐全，支持导入本地书籍，多端同步
Z-library 免费电子书下载
书童图书馆 论文下载
Notepad++ 文本类都能打开的阅读器
菜鸟教程 计算机相关教程网，根本学不完

思维导图及流程图

名称 推荐语
visio 流程图绕不开的软件，胜在标准
Excalidraw 功能强大的白班绘图工具，扩展性强。
Obsidian mindmap 在笔记中也能绘制思维导图
Mindmaster 7.3.1 版本就够用了。

Chat 类 AI

名称 推荐语
通义千问 用了快一年了，你就问吧，代码模式，ppt 模式真香
秘塔搜索 在线搜索强无敌

图片类

名称 推荐语
Adobe Photoshop 学了不亏
Honeyview 小巧的图片阅读器
佐糖 网页图片处理工具
Picasa 3 Google 图片管理软件，已停止更新，并不妨碍其好用
Unsplash 开源图片下载网站
Pixabay 开源图片下载网站
Pexels 开源图片下载网站
拾光壁纸 最好用的壁纸应用，美图查看器，Windows 商店可下载

IDE

名称 推荐语
VS code 绕不开的 IDE 扩展强大，支持 jupyter notebook
Pycharm 专业的 IDE, 智能提示和纠错
Cursor 智能 IDE

地图 GIS

名称 推荐语
Omap 奥维地图，简单好用，功能强大
QGIS 开源，界面有好，适合新手的第一款 GIS
Arcgis/Arcgis pro 专业、且永远绕不开的 GIS 软件

BI

名称 推荐语
Tableau BI 界的 office
PowerBI BI 界的 wps
