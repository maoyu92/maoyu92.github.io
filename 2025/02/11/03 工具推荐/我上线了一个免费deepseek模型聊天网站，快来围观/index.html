快来围观，我上线了一个免费AI对话网站

帮你省流，免费的是 deepseek 蒸馏模型。

利用 streamlit + 硅基流动 +阿里云服务器，做了个对话网站。大家可以轻松体验蒸馏的deepseek、通义千问、 llama 3、零一万物等，不需要 api key ，因为用的我的~，当然你也可以用自己的 API key。

我的费用就是自己的服务器，为爱发电，如果喜欢，给我买杯咖啡 ~
