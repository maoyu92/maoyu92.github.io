word、ppt 接入deepseek 文章补充说明（免费送代码）

再次感谢大家的关注，针对前面两篇文章进行一下补充。

一、宏安全问题

很多人说红安全问题一直报错，原因一是没有在信任中心中开启宏安全设置。

二是宏只保存在当前的 word 下，没有放在模板文件下，docx 是不支持宏的，需要在 dotm 中。

那么如果我们将宏保存在模板中，他每次新建都会套用模板，就不存在问题了。

三是版本问题
有的朋友，始终不支持宏，那么我建议重装 office，至少能够录制宏，再来操作。

二、文件下载与打开问题

我之前使用的蓝奏云，是为了大家方便，且不用输入验证码。代码免费下载。

有的朋友反应下不了，我决定优化到百度网盘里。
另外我直接将宏导出为 abs 文件，需要的朋友自己导入就可以了，而且我新增了翻译功能。

更新后，为了保持一致，后台回复不变。
回复 wordAI 、1001，以此获得对应的代码。

新增回复 1000 ，获得所有代码和模板文件。ABS 文件只需要在模块中选择导入就行。

三、PPT 宏安全问题

有的朋友反应，代码第一次能用，第二次不能用，道理和 word 模板一样，ppt 默认 vb 编辑界面没有模板，需要自己建一个 potm 模板文件，然后用这个模板右键打开，就可以使用了。

四、代码修改

需要修改的得放就是 API key ，url ， model name。不会的将代码和自己的 apikey 为给 deepseek ，让他帮你改。

我默认用的硅基流动的蒸馏版，已经够用了，如果你要用满血 deepseek ，请用自己的 api key。如果不是硅基流动的，需要替换 API key ，url ， model name。再对应的 api 文档中查找就行。

如果你有更多想法，比如接入 excel，可以自行尝试，我有新发现也会发表新的文章，欢迎关注~

五、关于 wps

Wps 在 vb 模式下，代码一模一样，但是默认没有 vb 模式，需要安装 vb 插件或者用企业版。
至于 js, 我并不熟悉，有大佬可以补充下。
